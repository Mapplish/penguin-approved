# 🐧 Penguin Approved

> Because 31°C Vorlauf is a lifestyle.

Penguin Approved is an open-source Home Assistant project for optimizing
underfloor heating and heat pumps using real data instead of fixed assumptions.

The project aims to:

- 🔥 optimize heating
- ❄️ optimize cooling
- ⚡ reduce power consumption
- 📈 visualize everything
- 🧠 learn the thermal behavior of a house
- 🏠 automatically improve hydraulic balancing

No cloud.
No subscriptions.
Just data.

🐧 Penguin Approved – Development started on 2026-07-22.
